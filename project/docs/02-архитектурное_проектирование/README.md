# Архитектурное проектирование

## Этап 2 (Недели 5-6, 10%)

### Основные задачи:

1. **Архитектурная диаграмма**
   - Слоистая архитектура PCMEF
   - Компонентная структура
   - Взаимодействие компонентов

2. **Технологическая архитектура**
   - Java 17 + Spring Boot 4.0.6
   - PostgreSQL
   - Maven (WAR)
   - Thymeleaf + Bootstrap 5
   - Spring Security

3. **Детализация пакетов**
   - `com.skfu.project.control` — контроллеры
   - `com.skfu.project.mediator` — бизнес-логика
   - `com.skfu.project.entity` — сущности
   - `com.skfu.project.foundation` — репозитории, инфраструктура

### Примечания:

- Архитектура должна строго следовать PCMEF (Presentation → Control → Mediator → Entity → Foundation)
- Все зависимости должны идти сверху вниз (циклические запрещены)
- Технологический стек соответствует современным требованиям (Spring Boot 4.0.6)
