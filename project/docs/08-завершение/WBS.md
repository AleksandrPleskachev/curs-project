## Уровень 4: Конкретные действия

### Пример детализации задачи 6.1.1 (Создание класса Project)

1. Создать пакет `com.skfu.project.entity`
2. Создать класс `Project` в пакете `entity`
3. Добавить аннотацию `@Entity`
4. Добавить поле `id` с аннотацией `@Id` и `@GeneratedValue`
5. Добавить поля: `name`, `description`, `startDate`, `endDate`, `budget`, `status`
6. Добавить геттеры и сеттеры
7. Добавить конструкторы
8. Создать репозиторий `ProjectRepository`
9. Создать mediator `ProjectMediator`
10. Создать controller `ProjectController`

### Пример детализации задачи 7.2.1 (Настройка dependency injection)

1. Проверить наличие `@Service` в медиаторах
2. Проверить наличие `@Repository` в репозиториях
3. Проверить наличие `@Controller` в контроллерах
4. Проверить конструкторную инъекцию в контроллерах
5. Проверить конструкторную инъекцию в медиаторах
