## 2. Требования к функциональности

### 2.1 Модуль управления проектами

**Описание:** Управление строительными проектами от создания до завершения.

**Функции:**
- Создание проекта
- Чтение списка проектов
- Обновление данных проекта
- Удаление проекта
- Фильтрация по статусу, датам, бюджету

**REST API:**
- `GET /api/projects` - список проектов
- `GET /api/projects/{id}` - получение проекта
- `POST /api/projects` - создание проекта
- `PUT /api/projects/{id}` - обновление проекта
- `DELETE /api/projects/{id}` - удаление проекта

## 9. Архитектура

### 9.1 PCMEF Architecture

```
Presentation (Web Controllers)
    ↓
Control (MVC Controllers)
    ↓
Mediator (Business Logic)
    ↓
Entity (Domain Models)
    ↓
Foundation (Repositories, Infrastructure)
```

### 9.2 Пакетная структура

- `com.skfu.project.control` - контроллеры
- `com.skfu.project.mediator` - бизнес-логика
- `com.skfu.project.entity` - сущности
- `com.skfu.project.foundation` - репозитории
- `com.skfu.project.presentation` - web-контроллеры
