# Матрица стейкхолдеров

| Роль | Интерес | Влияние | Ожидания |
| :--- | :--- | :--- | :--- |
| **Руководство компании** | Повышение эффективности, снижение издержек, рост прибыли. | Высокое | Точная отчетность, контроль над проектами, прогнозирование. |
| **Менеджеры по работе с клиентами** | Упрощение взаимодействия с заказчиками, быстрое оформление заявок. | Среднее | Удобный интерфейс для ввода данных о клиентах и проектах. |
| **Инженеры-сметчики** | Автоматизация расчетов, доступ к актуальным ценам на материалы. | Среднее | Инструменты для быстрого и точного составления смет. |
| **Руководители проектов** | Контроль сроков, бюджета, распределение задач. | Высокое | Панель управления проектом, отслеживание прогресса, управление ресурсами. |
| **Бухгалтерия** | Точность финансовых данных, своевременное выставление счетов. | Среднее | Интеграция с данными по договорам и оплатам, формирование финансовых отчетов. |
| **Заказчики** | Прозрачность хода работ, своевременная сдача объекта. | Низкое (но критично для репутации) | Возможность отслеживать прогресс своего проекта онлайн. |
| **Сотрудники (исполнители)** | Четкое понимание своих задач и сроков. | Низкое | Доступ к списку своих задач и инструкциям. |
| **Администратор ИС** | Стабильная работа системы, безопасность данных. | Высокое | Надежная и защищенная система с возможностью управления пользователями. |
