## Инструкция по генерации диаграмм

1. Откройте [PlantUML Live](https://www.planttext.com/)
2. Скопируйте PlantUML-код из файлов `*.puml` в папке `docs/00-инициация/`
3. Вставьте код в редактор PlantUML Live и нажмите "Generate"
4. Сохраните полученные изображения в формате PNG в папку `docs/images/`
