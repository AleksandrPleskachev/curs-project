```
+--------------------------------------------------+
| Профиль пользователя                            |
+--------------------------------------------------+
| [Имя пользователя]                             |
| Роль: ROLE_USER                                |
| Email: user@example.com                        |
|                                                 |
| === Мои задачи ===                              |
| ┌────────────────────────────────────────────┐  |
| │ [Название задачи] [Высокий]               │  |
| │ Статус: [В работе ▼]                     │  |
| │ Срок: 01.06.2026 - 30.06.2026             │  |
| └────────────────────────────────────────────┘  |
|                                                 |
| === Мои проекты ===                             |
| ┌────────────────────────────────────────────┐  |
| │ Название проекта 1                        │  |
| │ Роль: Инженер                             │  |

```

## Примечания

- Все формы содержат обязательные поля (marked с *)
- Дата выбирается через datepicker
- Списки с фильтрацией и автодополнением
- Статусы отображаются цветами (зеленый - завершен, желтый - в работе, красный - критический)
- Уведомления (toast) для успешных действий и ошибок
- Модальные окна для подтверждения удаления
