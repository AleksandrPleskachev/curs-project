# Спецификация методов

## ProjectMediator

### `Project createProject(Project project)`
*   **Назначение:** Создает новый проект в системе. Выполняет валидацию данных и сохраняет проект в базу данных.
*   **Параметры:** `project` - объект сущности Project, содержащий данные для создания.
*   **Возвращаемое значение:** Объект Project с присвоенным ID.
*   **Исключения:** `IllegalArgumentException` - если данные проекта некорректны (например, бюджет <= 0, дата начала позже даты окончания).
*   **Побочные эффекты:** Сохраняет проект в таблицу `project`.

### `Project findProjectById(Long id)`
*   **Назначение:** Находит проект по его идентификатору вместе со всеми связанными сущностями (задачи, сметы, договор).
*   **Параметры:** `id` - идентификатор проекта.
*   **Возвращаемое значение:** Объект Project, если найден.
*   **Исключения:** `EntityNotFoundException` - если проект с таким ID не существует.
*   **Побочные эффекты:** Выполняет запросы к таблицам `project`, `task`, `estimate`, `contract`.

## EstimateMediator

### `Estimate approveEstimate(Long id)`
*   **Назначение:** Утверждает смету по ее идентификатору. Меняет статус сметы и инициирует создание задачи по запуску проекта.
*   **Параметры:** `id` - идентификатор сметы.
*   **Возвращаемое значение:** Объект Estimate с обновленным статусом.
*   **Исключения:** `EntityNotFoundException` - если смета не найдена; `IllegalStateException` - если смета уже утверждена или отклонена.
*   **Побочные эффекты:** Обновляет запись в таблице `estimate`, создает новую запись в таблице `task`.

## TaskMediator

### `Task createTask(Task task)`
*   **Назначение:** Создает новую задачу для проекта.
*   **Параметры:** `task` - объект сущности Task.
*   **Возвращаемое значение:** Объект Task с присвоенным ID.
*   **Исключения:** `IllegalArgumentException` - если данные задачи некорректны.
*   **Побочные эффекты:** Сохраняет задачу в таблицу `task`.
